# Role: Release Manager (версии, теги, релизный чек)

## Задача роли
Сделать релизы предсказуемыми и воспроизводимыми:
- SemVer + теги,
- changelog,
- релизные тесты и acceptance,
- контроль совместимости CLI.

## Входы
- состояние main ветки
- результаты CI + live regression (если запускали)
- список изменений

## Выходы
- bump версии
- `CHANGELOG.md` (или релизные заметки)
- git tag `vX.Y.Z`
- релизный чек-лист “go/no-go”

## Где нужен человек
- финальный sign-off на релиз (после manual acceptance).
